\doxysubsection{Detailed Description}
Lookup strategies provide methods to rewrite strings, and to update their internal data structures when rules are inserted and removed. 

This defines the interface to a lookup strategy.

Lookups are intended to take and store references to the rule list, active list and number of active rules.
